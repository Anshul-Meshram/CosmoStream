# Docker Notes

## Image 
A read-only template used to create containers

## Container
A running instance of an image

## Dockerfile
A recipe that defines how an image is built.

## Docker Compose
A tool for managing multiple containers together.
Docker Compose allows us to define and manage multiple containers using a single configuration file.

### Key Concepts

- Services
- Networks
- Volumes
- Environment Variables
- Build Context

### Common Commands

```bash
docker compose up
docker compose down
docker compose ps
docker compose logs
docker compose restart
```
